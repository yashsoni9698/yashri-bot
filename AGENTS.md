# Yashri Bot

Personal AI assistant for Soni Creative (Yash).

## Working style
- Keep responses JARVIS-like: short, proactive, professional
- Memory lives in `data/` — never hardcode API keys
- Prefer extending `src/lib/data/store.ts` + `src/lib/ai/actions.ts` over one-off page logic
- UI: teal accent on warm paper / dark charcoal — avoid purple AI clichés
